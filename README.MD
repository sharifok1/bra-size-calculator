A simple and accurate bra size calculator for wordpress based on Bangladesh standard sizing rules.

== Description ==

**Bra Size Calculator for wordpress plugin** is a lightweight WordPress plugin that helps users calculate their correct bra size using standard measurement rules commonly used in Bangladesh.

The plugin provides a clean frontend calculator via shortcode and allows basic customization from the WordPress admin panel.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory  
2. Activate the plugin through the **Plugins** menu in WordPress  
3. Go to **Settings â†’ Bra Size Calculator** to configure options  
4. Use the shortcode to display the calculator on any page or post  

== Usage ==

Add the following shortcode where you want the calculator to appear:

[brasical_size_calculator]
